## Overview

The `array-intersection` transformation finds the intersection (common elements) of two or more arrays.

## Syntax

```text
array-intersection(<variable 1>, <variable 2>)
```

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `variable 1` | array | Yes | First array |
| `variable 2` | array | Yes | Second array |

## Examples

```text
array-intersection(<variable 1>, <variable 2>)
```

<Note>
  Returns an array containing only the elements that appear in both arrays.
</Note>
