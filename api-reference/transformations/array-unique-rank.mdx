## Overview

The `array-unique-rank` transformation returns unique values from an array along with their ranking.

## Syntax

```text
array-unique-rank(<variable>)
```

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `variable` | array | Yes | The array to process |

## Examples

```text
array-unique-rank(<variable>)
```
