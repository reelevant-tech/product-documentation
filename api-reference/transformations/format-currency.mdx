## Overview

The `format-currency` transformation formats a number as a currency value.

## Syntax

```text
format-currency(<variable>)
```

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `variable` | number | Yes | The number to format as currency |

## Examples

```text
format-currency(<variable>)
```

<Info>
  For more advanced currency formatting with locale and currency code support, see [Format Currency with Locale](/api-reference/transformations/format-currency-locale) or [Format Currency with Locale and Currency](/api-reference/transformations/format-currency-locale-currency).
</Info>
