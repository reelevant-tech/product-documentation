## Overview

The `length` transformation returns the number of characters in a string.

## Syntax

```text
length(<variable>)
```

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `variable` | string | Yes | The string to measure |

## Examples

```text
length(<variable>)
```
