## Overview

The `round` transformation rounds a number to a specified number of decimal places.

## Syntax

```text
round(<variable>, decimals=0)
```

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `variable` | number | Yes | - | The number to round |
| `decimals` | number | No | `0` | Number of decimal places |

## Examples

```text
round(<variable>, decimals=2)
round(<variable>, 0)
```
