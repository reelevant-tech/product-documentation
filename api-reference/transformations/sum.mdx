## Overview

The `sum` transformation adds two numbers together.

## Syntax

```text
sum(<variable 1>, <variable 2>)
```

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `variable 1` | number | Yes | First number to add |
| `variable 2` | number | Yes | Second number to add |

## Examples

```text
sum(<variable 1>, <variable 2>)
```
