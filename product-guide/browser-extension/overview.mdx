## What is the Browser Extension?

The Reelevant Browser Extension is a Chrome extension that helps you integrate [workflows](/platform-guide/workflows/overview) directly with your website. It provides a side panel for browsing workflows and configuring integrations, plus a content script that lets you select DOM elements on any page for targeting.

## Key Features

| Feature | Description |
|---------|-------------|
| **Side panel** | Browse your workflow list, view workflow details, and configure integrations. |
| **DOM selector** | A visual toolbox overlaid on the page to pick HTML elements for integration targeting. |
| **Company picker** | Switch between companies if you have access to multiple tenants. |

## Installation & Setup

<Steps>
  <Step title="Install the extension">
    Install the Reelevant extension from the Chrome Web Store. Once installed, the Reelevant icon appears in your browser toolbar.
  </Step>
  <Step title="Log in">
    Click the extension icon to open the side panel. Enter your Reelevant credentials:

    | Field | Description |
    |-------|-------------|
    | **Email** | Your Reelevant account email. |
    | **Password** | Your Reelevant account password. |

    Click **Sign in** to authenticate.
  </Step>
  <Step title="Select a company">
    If your account has access to multiple companies, a company picker appears. Select the company you want to work with.
  </Step>
  <Step title="Start using the extension">
    Once logged in, the side panel shows your workflow list. Navigate to the target website to start configuring integrations.
  </Step>
</Steps>
