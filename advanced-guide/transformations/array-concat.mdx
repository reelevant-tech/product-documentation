## Overview

The `concat` transformation concatenates two arrays into a single array.

## Syntax

```text
concat(<variable 1>, <variable 2>)
```

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `variable 1` | array | Yes | First array |
| `variable 2` | array | Yes | Second array |

## Examples

```text
concat(<variable 1>, <variable 2>)
```
