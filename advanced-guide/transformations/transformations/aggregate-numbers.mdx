## Overview

The `aggregate-numbers` transformation performs aggregation operations on an array of numbers.

## Syntax

```text
aggregate-numbers(<variable>, type="sum")
```

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `variable` | array | Yes | Array of numbers to aggregate |
| `type` | string | Yes | Aggregation type: `"sum"`, `"average"`, or `"median"` |

## Examples

```text
aggregate-numbers(<variable>, type="sum")
aggregate-numbers(<variable>, type="average")
aggregate-numbers(<variable>, type="median")
```
