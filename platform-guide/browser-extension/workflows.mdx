## Workflow List

After logging in, the side panel shows your [workflows](/platform-guide/workflows/overview) in a searchable list:

| Control | Description |
|---------|-------------|
| **Search** | Filter workflows by name. |
| **Workflow cards** | Each workflow shows its name, status, and channel type. Click to view details. |

## Workflow Detail

Click a workflow to see its details:

| Section | Description |
|---------|-------------|
| **Workflow info** | Name, status, and metadata. |
| **Channels** | The [channels](/platform-guide/workflows/channels) configured for this workflow. |
| **Integration** | Navigate to the [integration configuration](/platform-guide/browser-extension/integration) for the selected channel. |

## Navigation

| Control | Description |
|---------|-------------|
| **Back button** | Return to the workflow list. |
| **Settings** | Open extension settings (e.g., logout, company picker). |
| **Refresh** | Refresh the workflow list. |
