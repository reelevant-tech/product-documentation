The Loyalty Programme category covers challenges around driving enrolment, increasing engagement within the programme, and maximising the value generated by loyalty members.

## Key Challenges in this Category

<CardGroup cols={2}>

<Card title="Loyalty Programme Recruitment" icon="user-plus">
A loyalty programme member generates over 30% more value than a non-member. Brands adapt their strategy to encourage existing buyers and prospects to join the programme.

**Connected Use Cases:** 3
</Card>

<Card title="Prospect Loyalty Recruitment" icon="id-card">
Convincing prospects (not yet customers) to join the loyalty programme before their first purchase. Immediate value proof and benefit simulation are powerful conversion levers.

**Connected Use Cases:** 2
</Card>

<Card title="Programme Status Progression" icon="stairs">
Reducing the time to move from one tier to the next by making progression tangible and motivating. Real-time visualisation of points, tiers, and benefits to unlock creates a strong engagement effect.

**Connected Use Cases:** 3
</Card>

<Card title="Points Utilisation Maximisation" icon="coins">
Encouraging members to use their points before expiration and to accumulate them through diverse behaviours. Unused points represent dormant value and a re-engagement opportunity.

**Connected Use Cases:** 2
</Card>

<Card title="Engagement via Challenges & Missions" icon="trophy">
Increasing participation in challenges and missions to create regular non-transactional engagement. Gamification reinforces brand preference and interaction frequency beyond purchase acts.

**Connected Use Cases:** 1
</Card>

</CardGroup>

## Measuring Incremental Value

For loyalty programme challenges, the key formulas are:

```
Uplift enrolment = Rate_exposed − Rate_control
Incremental Value = Incremental enrolments × (Annual value_member − Annual value_non-member)
```

See the full worked example in [Key Challenges — Challenge 2](/product-adoption/incremental-value/key-challenges#challenge-2--loyalty-programme-enrolment).
