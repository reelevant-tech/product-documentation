## Vue d'ensemble

Le Live Polling est un Data Node sans configuration.

Il retourne le pourcentage actuel de clics attribués à son Branch.

La valeur est calculée au moment de l'ouverture, elle reflète donc l'instant où l'image est récupérée.

## Comportement

| Élément | Description |
|---|---|
| Sortie | Un nombre. |
| Configuration | Aucune. |
| Comptage | Chaque utilisateur unique est compté une fois par Branch. |
| Calcul | Clics sur ce Branch ÷ total des clics du sondage × 100. |
| Libellé dans l'éditeur | Pourcentage actuel (en direct) de clics. |

## Connexion au Content Editor

Vous l'ajoutez dans le Content Editor via **Workflow → Live polling**.

Le preset Live poll ajoute une image et un composant texte.

Le composant texte est lié au pourcentage en direct et ajoute un signe de pourcentage.

C'est un pattern courant pour les visuels de votes et sondages en direct dans les emails.
