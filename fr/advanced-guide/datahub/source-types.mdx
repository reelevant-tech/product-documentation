## Présentation

Lors de la [création d'une Datasource](/fr/product-guide/datahub/creating-a-datasource), vous sélectionnez un **type de source** qui définit comment Reelevant récupère les données. Chaque type de source possède ses propres champs de configuration et capacités.

## Formats de fichiers pris en charge

Les connecteurs basés sur des fichiers (URL, File Upload, FTP/SFTP, GCS, S3) détectent automatiquement le format du fichier. Les formats suivants sont pris en charge sur toutes les sources basées sur des fichiers :

| Format | Description |
|--------|-------------|
| **CSV** | Valeurs séparées par des virgules. Le délimiteur est détecté automatiquement (virgule, point-virgule, tabulation, pipe). Supporte les [noms de colonnes explicites](/fr/advanced-guide/datahub/csv-options) pour les fichiers sans en-tête. |
| **JSON** | Fichiers JSON standards avec un tableau ou objet racine. |
| **NDJSON** | JSON délimité par des retours à la ligne (un objet JSON par ligne). |
| **XML** | Fichiers XML — le chemin de l'élément racine est détecté automatiquement. |
| **Parquet** | Format columaire Apache Parquet. |
| **Avro** | Format de sérialisation Apache Avro. |
| **XLSX** | Fichiers Microsoft Excel. |

<Info>
  Les fichiers compressés (`.gz`, `.zip`) sont automatiquement décompressés avant le parsing pour toutes les sources basées sur des fichiers.
</Info>

<Info>
  Les connecteurs basés sur des fichiers (URL, FTP/SFTP, GCS, S3) supportent également le [déchiffrement PGP](/fr/advanced-guide/datahub/pgp-decryption) — les fichiers chiffrés (`.asc`, `.pgp`, `.gpg`) sont déchiffrés de manière transparente lors de l'import lorsque `pgpPrivateKey` est configuré.
</Info>

## Sources multiples

Certains types de Datasources supportent les **sources multiples**. Vous pouvez ajouter plusieurs configurations de source (par ex. plusieurs URLs ou chemins FTP) à une seule Datasource en utilisant le bouton **Add a source**. Chaque source est traitée indépendamment, et les résultats sont combinés.

Utilisez **Delete the source** pour supprimer des configurations de source individuelles.
