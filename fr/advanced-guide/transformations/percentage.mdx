description: 'Formater une valeur en pourcentage'
---

## Présentation

La transformation `percentage` multiplie une valeur décimale par 100 et la formate sous forme de chaîne de pourcentage avec le nombre de décimales spécifié.

## Syntaxe

```text
percentage(<variable>, decimal=2)
```

## Paramètres

| Paramètre | Type | Requis | Description |
|-----------|------|--------|-------------|
| `variable` | number | Oui | La valeur décimale à convertir en pourcentage (par ex. 0.75 devient 75%) |
| `decimal` | number | Oui | Nombre de décimales dans le résultat |

## Exemples

```text
percentage(<variable>, decimal=2)
percentage(<variable>, decimal=0)
```
