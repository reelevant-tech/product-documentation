---

## Présentation

La transformation `unique` renvoie un tableau contenant uniquement les valeurs uniques, en supprimant les doublons.

## Syntaxe

```text
unique(<variable>)
```

## Paramètres

| Paramètre | Type | Requis | Description |
|-----------|------|--------|-------------|
| `variable` | array | Oui | Le tableau dont on souhaite extraire les valeurs uniques |

## Exemples

```text
unique(<variable>)
```
