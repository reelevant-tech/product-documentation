description: 'Convertir une chaîne en nombre'
---

## Présentation

La transformation `parse-number` convertit une valeur de type chaîne en nombre.

## Syntaxe

```text
parse-number(<variable>)
```

## Paramètres

| Paramètre | Type | Requis | Description |
|-----------|------|--------|-------------|
| `variable` | string | Oui | La valeur chaîne à convertir en nombre |

## Exemples

```text
parse-number(<variable>)
```

<Note>
  Cette transformation tentera de convertir la chaîne en nombre. Si la chaîne ne peut pas être convertie, elle peut renvoyer une erreur ou une valeur nulle.
</Note>
