description: 'Convertir une chaîne en nombre'
---

## Vue d'ensemble

La transformation `parse-number` convertit une valeur de chaîne en nombre.

## Syntaxe

```text
parse-number(<variable>)
```

## Paramètres

| Paramètre | Type | Requis | Description |
|-----------|------|----------|-------------|
| `variable` | string | Oui | La valeur de chaîne à analyser en nombre |

## Exemples

```text
parse-number(<variable>)
```

<Note>
  Cette transformation tentera d'analyser la chaîne en nombre. Si la chaîne ne peut pas être analysée, elle peut renvoyer une erreur ou une valeur null.
</Note>
