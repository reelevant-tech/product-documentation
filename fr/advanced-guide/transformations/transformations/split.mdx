## Vue d'ensemble

La transformation `split` divise une chaîne en tableau à partir d'un délimiteur.

## Syntaxe

```text
split(<variable>, delimiter=",")
```

## Paramètres

| Paramètre | Type | Requis | Description |
|-----------|------|----------|-------------|
| `variable` | string | Oui | La chaîne à diviser |
| `delimiter` | string | Oui | Le caractère ou la chaîne délimiteur |

## Exemples

```text
split(<variable>, delimiter=",")
split(<variable>, delimiter=" ")
```
