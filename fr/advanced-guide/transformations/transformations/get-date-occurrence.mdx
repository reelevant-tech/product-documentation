## Vue d'ensemble

La transformation `get-date-occurence` trouve la prochaine (ou la plus proche) occurrence d'une date dans l'année en cours ou l'année suivante. C'est utile pour les événements récurrents comme les anniversaires.

## Syntaxe

```text
get-date-occurence(<variable>)
```

## Paramètres

| Paramètre | Type | Requis | Description |
|-----------|------|----------|-------------|
| `variable` | date | Oui | La date de référence |
| `inputLocale` | string | Non | Locale utilisée pour analyser la date d'entrée : `"pt"`, `"nl"`, `"en"`, `"de"`, `"ja"`, `"zh"` ou `"fr"` |
| `useNearest` | boolean | Non | Lorsque true, renvoie l'occurrence la plus proche (passée ou future) au lieu de toujours renvoyer la prochaine |

## Exemples

```text
get-date-occurence(<variable>)
get-date-occurence(<variable>, useNearest=true)
get-date-occurence(<variable>, inputLocale="fr")
```
