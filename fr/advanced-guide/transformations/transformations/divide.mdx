## Présentation

La transformation `divide` divise le premier nombre par le second nombre.

## Syntaxe

```text
divide(<variable 1>, <variable 2>)
```

## Paramètres

| Paramètre | Type | Requis | Description |
|-----------|------|--------|-------------|
| `variable 1` | number | Oui | Le dividende (nombre à diviser) |
| `variable 2` | number | Oui | Le diviseur (nombre par lequel diviser) |

## Exemples

```text
divide(<variable 1>, <variable 2>)
```

<Warning>
  La division par zéro provoquera une erreur. Assurez-vous que le diviseur n'est pas nul.
</Warning>
