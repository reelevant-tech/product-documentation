## Vue d'ensemble

La transformation `replace` remplace les occurrences d'une sous-chaîne au sein d'une chaîne.

## Syntaxe

```text
replace(<variable>, search="", isRegex=false, replace="")
```

## Paramètres

| Paramètre | Type | Requis | Description |
|-----------|------|----------|-------------|
| `variable` | string | Oui | La chaîne dans laquelle effectuer la recherche |
| `search` | string | Oui | La sous-chaîne ou le motif regex à trouver |
| `isRegex` | boolean | Oui | Indique si la chaîne de recherche est une expression régulière |
| `replace` | string | Oui | La chaîne de remplacement |

## Exemples

```text
replace(<variable>, search="old", isRegex=false, replace="new")
replace(<variable>, search="[0-9]+", isRegex=true, replace="X")
```

<Info>
  Pour remplacer par une valeur de variable, voir [Replace with Variable](/api-reference/transformations/replace-variable).
</Info>
