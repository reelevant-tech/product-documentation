## Vue d'ensemble

La transformation `length` renvoie le nombre de caractères d'une chaîne.

## Syntaxe

```text
length(<variable>)
```

## Paramètres

| Paramètre | Type | Requis | Description |
|-----------|------|----------|-------------|
| `variable` | string | Oui | La chaîne à mesurer |

## Exemples

```text
length(<variable>)
```
