## Présentation

La transformation `concat` concatène deux tableaux en un seul tableau.

## Syntaxe

```text
concat(<variable 1>, <variable 2>)
```

## Paramètres

| Paramètre | Type | Requis | Description |
|-----------|------|--------|-------------|
| `variable 1` | array | Oui | Premier tableau |
| `variable 2` | array | Oui | Second tableau |

## Exemples

```text
concat(<variable 1>, <variable 2>)
```
