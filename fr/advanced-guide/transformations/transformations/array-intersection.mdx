## Présentation

La transformation `intersection` trouve les éléments communs entre deux tableaux, en renvoyant uniquement les éléments présents dans les deux.

## Syntaxe

```text
intersection(<variable 1>, <variable 2>)
```

## Paramètres

| Paramètre | Type | Requis | Description |
|-----------|------|--------|-------------|
| `variable 1` | array | Oui | Premier tableau |
| `variable 2` | array | Oui | Second tableau |

## Exemples

```text
intersection(<variable 1>, <variable 2>)
```

<Note>
  Renvoie un tableau contenant uniquement les éléments présents dans les deux tableaux.
</Note>
