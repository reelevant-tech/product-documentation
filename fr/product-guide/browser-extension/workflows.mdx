## Liste des Workflows

Après vous être connecté, le panneau latéral affiche vos [Workflows](/fr/product-guide/workflows/overview) dans un tableau consultable :

| Colonne / Contrôle | Description |
|------------------|-------------|
| **Search** | Filtrez les Workflows par nom. |
| **Filter by channel** | Restreignez la liste aux Workflows d'un type de Channel spécifique. |
| **Name** | Nom du Workflow. Cliquez pour ouvrir les détails du Workflow. |
| **Status** | État actuel — **Draft** ou **Published**. |
| **Last update** | Date de la dernière modification du Workflow. |

## Détail d'un Workflow

Cliquez sur un Workflow pour voir ses détails :

| Section | Description |
|---------|-------------|
| **Workflow info** | Nom, statut et métadonnées. |
| **Channels** | Les [Channels](/fr/product-guide/workflows/channels) configurés pour ce Workflow. |
| **Instructions d'intégration** | Ouvrez les [instructions d'intégration](/fr/developer-docs/guides/browser-extension-integration) d'un Channel pour copier les URLs d'image et de redirection. |
| **Intégration on-site** | Pour les Channels côté client, configurez l'intégration sur votre site web à l'aide du [content script](/fr/developer-docs/guides/browser-extension-content-script). |

## Navigation

| Contrôle | Description |
|---------|-------------|
| **Bouton retour** | Revenez à la liste des Workflows. |
| **Settings** | Ouvrez les paramètres de l'extension (par exemple, déconnexion). |
| **Navigation inférieure** | Basculez entre les sections Workflows, Contents, DataHub et Analytics. |
