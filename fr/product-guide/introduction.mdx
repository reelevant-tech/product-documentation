## Bienvenue sur Reelevant

Reelevant est une plateforme de personnalisation qui permet aux équipes marketing de créer des expériences de contenu automatisées et individualisées sur tous les canaux — e-mail, web, mobile et landing pages.

La plateforme est organisée en cinq applications principales, accessibles depuis la barre de navigation latérale :

## Concepts clés

Avant de commencer, voici quelques termes que vous rencontrerez dans toute la plateforme :

| Concept | Description |
|---------|-------------|
| **Workflow** | Un graphe orienté de Nodes qui définit la logique de personnalisation d'une campagne. |
| **Content** | Un template visuel (bannière, section d'e-mail, etc.) conçu dans le Content Editor. |
| **Datasource** | Un flux de données externe (produits, emplacements, événements) importé dans la plateforme. |
| **[Channel](/fr/product-guide/workflows/channels)** | Le Node de départ d'un Workflow qui définit le canal de sortie (e-mail, web, mobile, landing page). |
| **Branch** | Un chemin d'exécution unique à travers un Workflow, depuis un [Channel](/fr/product-guide/workflows/channels) jusqu'à un [Output Node](/fr/product-guide/workflows/output-nodes). |
| **Team** | Une unité organisationnelle pour contrôler l'accès aux Workflows, Contents et Datasources. |
| **Variable Profile** | Un ensemble pré-configuré de paramètres d'URL (UTM, tracking) appliqué lors de l'intégration d'un Workflow. |

Pour une liste complète des termes, consultez le [Glossaire](/fr/glossary).

## Navigation

La plateforme utilise une barre latérale pour la navigation principale entre les applications. Au sein de chaque application, vous trouverez :

- **Pages de listing** — Des tableaux affichant tous vos éléments (Workflows, Contents, Datasources) avec recherche, filtres et actions groupées.
- **Pages d'édition** — Des éditeurs plein écran pour construire des Workflows et des templates de Content.
- **Pages de paramètres** — Des options de configuration spécifiques à chaque application.

Un **sélecteur de scope** en haut de la barre latérale vous permet de basculer entre différentes sociétés ou tenants si vous avez accès à plusieurs.
