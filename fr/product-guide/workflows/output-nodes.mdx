## Que sont les Output Nodes ?

Les Output Nodes sont les **Nodes terminaux** d'une Branch de Workflow. Chaque Branch qui commence par un [Channel](/fr/product-guide/workflows/channels) doit se terminer par un Output Node qui définit quel contenu est renvoyé à l'utilisateur.

## Types d'Output Nodes

<CardGroup cols={2}>
  <Card title="Visual Content" icon="paintbrush" href="/fr/product-guide/workflows/output-nodes/visual-content">
    Affiche un template visuel conçu dans le Content Editor.
  </Card>
  <Card title="JSON Template" icon="file-code" href="/fr/product-guide/workflows/output-nodes/json-template">
    Renvoie un payload JSON structuré basé sur un template prédéfini avec des variables dynamiques.
  </Card>
  <Card title="Empty" icon="square" href="/fr/product-guide/workflows/output-nodes/empty">
    Renvoie une réponse vide pour les groupes de contrôle ou les Branches de repli.
  </Card>
</CardGroup>

## Ajouter un Output Node

<Steps>
  <Step title="Ouvrir le sélecteur de sorties">
    Cliquez sur le bouton **Output** (icône contenu) dans la barre d'outils inférieure. Le panneau **Content types** s'ouvre.
  </Step>
  <Step title="Sélectionner un type de sortie">
    Choisissez **Visual content**, **JSON template** ou **Empty** parmi les options disponibles.
  </Step>
  <Step title="Connecter et configurer">
    Connectez l'Output Node à la fin d'une Branch. Si vous avez choisi Visual Content, sélectionnez le template de contenu depuis la bibliothèque de contenus.
  </Step>
</Steps>

## Conseils

- Chaque Branch doit se terminer par un Output Node pour que le Workflow soit valide et publiable.
- Un même Workflow peut combiner différents types de sorties entre ses Branches — par exemple, un Content sur la Branch principale et un Empty sur le groupe de contrôle.
- Lorsque vous sélectionnez un template de contenu, assurez-vous que les données disponibles dans la Branch correspondent aux variables dynamiques du template.
