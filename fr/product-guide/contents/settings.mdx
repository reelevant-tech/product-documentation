## Présentation

La page de paramètres du Content vous permet de gérer les ressources partagées entre tous vos templates de contenu. Accédez-y depuis le lien **Settings** dans la navigation latérale du Content Editor.

## Polices

L'onglet **Fonts** vous permet de gérer les ressources de polices personnalisées disponibles dans l'éditeur de contenu. Les polices personnalisées apparaissent aux côtés des polices système dans le sélecteur de famille de polices du composant [Text](/fr/product-guide/contents/components/text).

### Liste des polices

Le tableau des polices affiche toutes les polices personnalisées enregistrées. Si aucune police n'a été ajoutée, la page affiche « No fonts yet. Add your first font resource. »

### Ajouter une police

<Steps>
  <Step title="Cliquer sur Add font">
    Cliquez sur le bouton **Add font** pour ouvrir le formulaire de création de police.
  </Step>
  <Step title="Saisir le nom de la famille de polices">
    Tapez le nom de la **Font family** (par exemple, « Ma Police Personnalisée »). C'est le nom qui apparaîtra dans le sélecteur de polices.
  </Step>
  <Step title="Ajouter les fichiers de polices">
    Sous **Font files**, ajoutez une entrée par variante de police :

    | Champ | Description |
    |-------|-------------|
    | **Variant** | Sélectionnez la variante de poids (par exemple, Regular, Bold, Italic). |
    | **Font file** | Téléversez un fichier de police (.ttf, .otf, .woff, .woff2). |

    Cliquez sur **Add variant** pour ajouter des variantes de poids supplémentaires.
  </Step>
  <Step title="Enregistrer">
    Cliquez sur **Save** pour enregistrer la police. Elle sera immédiatement disponible dans le sélecteur de polices de l'éditeur de contenu.
  </Step>
</Steps>

### Modifier une police

Cliquez sur le bouton **Edit** à côté d'une police existante pour mettre à jour son nom de famille ou ajouter/supprimer des variantes.

<Info>
  Au moins une variante de fichier de police est requise par famille de polices.
</Info>
