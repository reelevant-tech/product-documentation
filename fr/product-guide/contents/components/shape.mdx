## Présentation

Les composants Shape vous permettent d'ajouter des éléments géométriques à vos templates de contenu. Trois types de formes sont disponibles :

| Forme | Description |
|-------|-------------|
| **Rectangle** | Une forme rectangulaire avec un rayon de bordure configurable. |
| **Circle** | Une forme circulaire. |
| **Triangle** | Un triangle équilatéral. |

Toutes les formes prennent en charge le remplissage, le contour et le style d'arrière-plan.

## Paramètres des Shapes

Lorsqu'une forme est sélectionnée, le panneau de configuration affiche les propriétés spécifiques au type de forme :

### Paramètres du Rectangle

| Propriété | Description |
|-----------|-------------|
| **Fill** | Couleur de remplissage (ou **No fill** pour transparent). |
| **Stroke** | Couleur du contour et épaisseur du trait (ou **No stroke**). |
| **Border radius** | Arrondi des coins en pixels. |
| **Background** | Couleur d'arrière-plan (ou **No background**). |
| **Line-weight** | Épaisseur du contour. |

### Paramètres du Circle

| Propriété | Description |
|-----------|-------------|
| **Fill** | Couleur de remplissage. |
| **Stroke** | Couleur du contour et épaisseur du trait. |
| **Background** | Couleur d'arrière-plan. |

### Paramètres du Triangle

| Propriété | Description |
|-----------|-------------|
| **Equilateral triangle** | Le type de triangle. |
| **Fill** | Couleur de remplissage. |
| **Stroke** | Couleur du contour et épaisseur du trait. |
| **Background** | Couleur d'arrière-plan. |

## Propriétés communes

Toutes les formes partagent :

| Propriété | Description |
|-----------|-------------|
| **Position** | Coordonnées X et Y sur le canvas. |
| **Taille** | Largeur et hauteur avec un bouton optionnel **Conserver le ratio d'aspect**. |
| **Rotation** | Angle de rotation en degrés. |
| **Lien de redirection** | URL vers laquelle les utilisateurs sont dirigés lorsqu'ils cliquent sur la forme. |
