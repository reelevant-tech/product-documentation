## Présentation

Le composant Image affiche des images statiques ou dynamiques dans votre template de Content. Les images peuvent provenir d'un téléversement direct, d'une URL collée ou d'une variable de [Datasource](/fr/product-guide/contents/data-binding) qui se résout en URL d'image au moment du rendu.

| Propriété | Valeur |
|-----------|--------|
| **Bouton de la barre d'outils** | Images |
| **Prend en charge le Data binding** | Oui — URL de l'image |
| **Prend en charge les Conditions** | Oui |

## Paramètres de l'Image

| Propriété | Description |
|-----------|-------------|
| **Image** | Téléversez un fichier image (PNG, JPG, GIF) ou configurez une URL. |
| **Variable** | Liez l'URL de l'image à une variable de Datasource à l'aide du sélecteur de variables. |

<Info>
  Seuls les formats PNG, JPG et GIF sont pris en charge pour le téléversement d'images.
</Info>

## Configuration

| Propriété | Description |
|-----------|-------------|
| **Position** | Coordonnées X et Y sur le canvas. |
| **Taille** | Largeur et hauteur avec un bouton optionnel **Conserver le ratio d'aspect**. |
| **Rotation** | Angle de rotation en degrés. |
| **Lien de redirection** | URL vers laquelle les utilisateurs sont dirigés lorsqu'ils cliquent sur l'image. |
