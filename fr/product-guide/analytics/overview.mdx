## Qu'est-ce qu'Analytics ?

Analytics est le centre de mesure et d'insights de la plateforme Reelevant. Il fournit des tableaux de bord prêts à l'emploi pour suivre la performance des campagnes, un outil flexible de data exploration pour les analyses personnalisées, et des requêtes enregistrées pour les rapports récurrents.

## Concepts clés

| Concept | Description |
|---------|-------------|
| **Dashboard** | Une vue prête à l'emploi présentant les métriques clés d'un aspect spécifique de vos campagnes (performance, audience, conversions, etc.). |
| **Measure** | Une métrique quantitative — impressions, clics, CTR, conversions, chiffre d'affaires, etc. |
| **Dimension** | Un attribut qualitatif utilisé pour regrouper ou segmenter les données — nom de Workflow, type d'appareil, nom de Content, etc. |
| **Analysis period** | La période des données affichées dans les tableaux de bord et les requêtes. |
| **Attribution** | Comment Reelevant crédite le chiffre d'affaires aux interactions avec le Content, via un [modèle d'attribution](/fr/product-guide/analytics/attribution) et une fenêtre rétrospective par type d'interaction. |
| **Individualized content** | Content diffusé via des Branches de [Workflow](/fr/product-guide/workflows/overview) personnalisées, par opposition au Content par défaut (non personnalisé). |
| **Uplift** | L'Incremental Value générée par la personnalisation par rapport à ce que le Content par défaut aurait obtenu. |
