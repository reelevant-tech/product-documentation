## Vue d'ensemble

Le tableau de bord Conversions suit les résultats commerciaux de vos campagnes — achats, chiffre d'affaires et comportement d'achat attribués aux interactions avec des Contents Reelevant.

## Métriques clés

| Métrique | Description |
|--------|-------------|
| **Total Conversion Count** | Nombre de conversions ayant suivi un clic sur un Content Reelevant. |
| **Unique Buyer Count** | Nombre de personnes différentes ayant acheté au moins une fois. |
| **Influenced Revenue** | Chiffre d'affaires crédité aux interactions avec le Content Reelevant, réparti sur le parcours client. |
| **Direct Revenue** | Chiffre d'affaires crédité uniquement à la dernière interaction Reelevant avant l'achat. |
| **Total Revenue** | Chiffre d'affaires généré par les conversions. |
| **Average Order Value** | Valeur moyenne des commandes pour les conversions. |
| **Average Revenue Per Buyer** | Chiffre d'affaires moyen généré par acheteur. |
| **Order Count Per Buyer** | Nombre moyen d'achats par acheteur. |
| **Time To Purchase From Click** | Temps moyen entre un clic sur un Content Reelevant et l'achat. |
| **Reference Count** | Nombre total de références produits dans les transactions. |
| **Average Reference Count Per Basket** | Nombre moyen de références produits par transaction. |
| **Conversion Rate** | Conversions par clic. |
| **Unique Conversion Rate** | Taux de conversion pour les individus uniques. |

## Comment fonctionne l'attribution

Reelevant crédite chaque conversion aux interactions avec le Content qui l'ont menée. Le crédit est partagé sur le parcours client selon votre **modèle d'attribution**, dans une **fenêtre rétrospective** propre à chaque type d'interaction. Configurez les deux dans les [Analytics Settings](/fr/product-guide/analytics/settings).

<Info>
  **Influenced Revenue** répartit le crédit sur chaque interaction éligible, tandis que **Direct Revenue** ne crédite que la dernière. Les comparer révèle la contribution de vos premières interactions. Voir [Attribution](/fr/product-guide/analytics/attribution) pour l'explication complète.
</Info>
