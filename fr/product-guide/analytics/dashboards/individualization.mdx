## Vue d'ensemble

Le tableau de bord Individualization mesure la part de votre diffusion de Content qui est personnalisée. Il compare le **Content individualisé** (diffusé via des [Branches de Workflow](/fr/product-guide/workflows/branches) personnalisées) au **Content par défaut** (contenus de repli non personnalisés).

## Métriques clés

| Métrique | Description |
|--------|-------------|
| **Individualized Opening Count** | Nombre d'ouvreurs ayant vu au moins un Content individualisé. |
| **Individualized Display Count** | Nombre d'affichages de Content individualisé. Un Content est considéré comme individualisé s'il résulte d'interactions de navigation. |
| **Individualized Display Rate** | Part du Content individualisé parmi tous les affichages de Content. |
| **Unique Individualized Opener** | Nombre d'ouvreurs uniques ayant vu du Content individualisé. |
| **Unique Individualized Opener Rate** | Ratio entre les ouvreurs individualisés et le total des ouvreurs. |

## Content individualisé vs. par défaut

Le tableau de bord inclut un tableau comparatif présentant côte à côte les métriques pour :

| Type | Description |
|------|-------------|
| **Individualized** | Content diffusé via des Branches de Workflow personnalisées. |
| **Default** | Content diffusé via des Branches de repli non personnalisées. |
| **Total** | Métriques combinées sur les deux types. |

<Info>
  Pour augmenter votre taux d'individualisation, ajoutez davantage de Branches personnalisées à vos [Workflows](/fr/product-guide/workflows/overview) à l'aide de [Data Nodes](/fr/product-guide/workflows/data-nodes) et de [conditions](/fr/product-guide/workflows/logic-nodes/conditions).
</Info>
