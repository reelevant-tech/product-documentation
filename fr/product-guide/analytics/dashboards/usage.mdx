## Vue d'ensemble

Le tableau de bord Usage suit votre consommation d'ouvertures pendant la période d'abonnement. Il vous aide à vérifier si votre usage reste dans le quota prévu.

## Aperçu de l'usage

La carte de consommation affiche :

| Champ | Description |
|-------|-------------|
| **Période d'abonnement** | Votre abonnement court de la date de début à la date de fin. |
| **Type de forfait** | Indique si le forfait comptabilise le **nombre d'ouvertures** ou le **nombre d'ouvreurs uniques**. |
| **Usage prévu** | Le quota de votre abonnement. |
| **Consommation actuelle** | La part du quota déjà utilisée. |

### Statut de consommation

Une bannière indique votre statut de consommation :

| Statut | Message |
|--------|---------|
| **Success** (vert) | Votre consommation est dans le quota prévu. |
| **Warning** (jaune) | Votre consommation a dépassé le quota prévu. |
| **Error** (rouge) | Votre consommation a largement dépassé le quota prévu. |

## Tableaux d'usage

Deux tableaux de décomposition fournissent des données de consommation détaillées :

| Tableau | Description |
|-------|-------------|
| **Usage by team** | Consommation détaillée par Team. |
| **Usage by channel** | Consommation détaillée par type de [Channel](/fr/product-guide/workflows/channels). |

<Tip>
  Utilisez la décomposition par Team pour identifier quels Teams génèrent le plus de consommation. Cela facilite la planification des capacités et l'allocation du budget.
</Tip>
