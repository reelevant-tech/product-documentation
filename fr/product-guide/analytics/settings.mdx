## Vue d'ensemble

La page Paramètres Analytics vous permet de configurer les paramètres Analytics globaux de votre entreprise. Accédez-y depuis le lien **Settings** dans la navigation latérale d'Analytics.

## Période d'attribution

La période d'attribution définit le nombre d'heures utilisé pour attribuer les conversions à une interaction avec un [Workflow](/fr/product-guide/workflows/overview). Lorsqu'un utilisateur clique sur un Content Reelevant puis effectue un achat dans cette fenêtre temporelle, la conversion est créditée à cette interaction avec le Content.

| Champ | Description |
|-------|-------------|
| **Attribution period (hours)** | La fenêtre rétrospective en heures (par exemple, 720 pour 30 jours). Laissez vide pour utiliser la période d'attribution par défaut. |

### Configurer la période d'attribution

1. Rendez-vous dans **Analytics > Settings**.
2. Saisissez le nombre d'heures souhaité dans le champ **Attribution period (hours)**.
3. Cliquez sur **Save**.

<Info>
  Modifier la période d'attribution affecte la façon dont les conversions sont comptabilisées dans le [tableau de bord Conversions](/fr/product-guide/analytics/dashboards/conversions) et les requêtes [Data Exploration](/fr/product-guide/analytics/data-exploration). Une période plus longue capte davantage de conversions mais peut inclure des achats moins directement attribuables.
</Info>
