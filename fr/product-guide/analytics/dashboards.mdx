## Vue d'ensemble

La page Tableaux de bord fournit des visualisations prêtes à l'emploi pour les données de vos campagnes. Chaque tableau de bord se concentre sur un aspect spécifique de la performance et peut être filtré par Workflow, campagne, catégorie de Branch et période.

## Filtres des tableaux de bord

Tous les tableaux de bord partagent une barre de filtres commune en haut :

| Filtre | Description |
|--------|-------------|
| **Workflows** | Sélectionnez un ou plusieurs Workflows pour filtrer les données. Recherchez par nom de Workflow. |
| **Campaigns** | Filtrez par campagne. |
| **Category** | Filtrez par catégorie de Branch. |
| **Analysis period** | Sélectionnez la période pour les données. |
| **Comparison period** | Activez éventuellement une période de comparaison avec une date de début personnalisée. |

<Tip>
  Cliquez sur **Explore data** pour ouvrir la page [Data Exploration](/fr/product-guide/analytics/data-exploration) avec les filtres actuels déjà appliqués.
</Tip>
