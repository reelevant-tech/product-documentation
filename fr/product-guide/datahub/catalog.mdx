## Vue d'ensemble

Le Catalog est le point de départ pour ajouter de nouvelles Datasources. Il présente une galerie d'intégrations disponibles organisées par catégorie. Chaque carte affiche une brève description de l'intégration et un bouton **Connect** pour lancer l'assistant de configuration.

## Parcourir le Catalog

Utilisez les contrôles en haut du catalog pour trouver l'intégration dont vous avez besoin :

| Contrôle | Description |
|---------|-------------|
| **Filtre par catégorie** | Filtrez par catégorie — All categories, Product, Location, Generic, Analytics, Social. |
| **Search** | Recherchez par nom d'application pour trouver rapidement une intégration précise. |

## Intégrations disponibles

### Product

| Intégration | Description |
|-------------|-------------|
| **Product** | Importez vos données produits depuis un fichier ou une URL. |
| **Shopify Products** | Connectez-vous à votre Shopify et récupérez tous les produits. |
| **Best Products** | Calculez quel produit est le meilleur selon un événement de comportement précis. |
| **Cross sell** | Calculez un cross sell basé sur une stratégie d'achats conjoints. |
| **Top product attributes** | Calculez les meilleures marques/catégories pour chaque utilisateur. |
| **Multi-country product datasources** | Créez une Datasource avec un flux produits par pays. |
| **Lengow** | Importez votre catalogue de produits depuis Lengow. |
| **DataFeedWatch** | Récupérez les produits directement depuis votre boutique DataFeedWatch. |
| **Merge two datasources** | Ajoutez les données d'une autre Datasource aux données produits. |

### Location

| Intégration | Description |
|-------------|-------------|
| **Location** | Importez une liste de lieux depuis un fichier ou une URL. |
| **Google My Business** | Connectez-vous à Google My Business pour importer et synchroniser vos magasins. |
| **Partoo** | Importez et synchronisez les magasins depuis votre compte Partoo. |

### Generic

| Intégration | Description |
|-------------|-------------|
| **Generic** | Importez des données arbitraires depuis un fichier ou une URL. |
| **BigQuery** | Synchronisez des données d'une table BigQuery vers Reelevant. |
| **Snowflake** | Synchronisez des tables de Snowflake vers Reelevant. |
| **Google Sheets** | Importez des données depuis une Google Sheet publique. |
| **Editable datasource** | Datasource modifiable directement sur la plateforme. |

### Analytics

| Intégration | Description |
|-------------|-------------|
| **Website events** | Stockez le comportement des utilisateurs de votre site web dans Reelevant. |
| **Google Tag Manager** | Intégrez GTM pour stocker le comportement des utilisateurs de votre site web. |
| **Shopify Website Events** | Connectez directement les événements de site web de votre boutique Shopify. |
| **Customer purchases** | Achats cross-canal (en ligne et hors ligne). |
| **Customer data** | Données clients synchronisées depuis un CRM. |

<Info>
  Certaines intégrations peuvent ne pas être disponibles selon votre formule d'abonnement. Contactez votre Customer Success Manager pour accéder à davantage d'intégrations.
</Info>

## Connecter une Datasource

Cliquez sur **Connect** sur n'importe quelle carte du catalog pour lancer l'[assistant de création](/fr/product-guide/datahub/creating-a-datasource). L'assistant vous guide à travers la configuration de la source, le field mapping et la validation.
